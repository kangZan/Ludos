# Changelog

## v0.1.0 (2026-02-05)

### Added
- 初始版本
- 三阶段架构：初始化 → 自主推演循环 → 润色输出
- 主持人智能体：场景播报、回合管理、客观记录、回合评估
- 角色智能体：基于档案的自主决策、第一人称视角
- 润色智能体：文学化叙事生成
- 认知红绿灯机制（三层信息隔离防御）
- 秘密压力值系统（关键词匹配 + 衰减 + 阈值警告）
- "下一角色决定结果"的角色驱动叙事模式
- 非数值化冲突裁决
- LangGraph StateGraph 状态管理
- InMemorySaver / AsyncPostgresSaver 状态持久化
- Human-in-the-loop 中断/恢复支持
- 流式输出 (astream)
- CLI 入口
- 结构化日志 (structlog)
- 单元测试覆盖

### Planned
- 角色对外交互新增“小动作”字段
- 主持人维护小动作记忆并判定是否被其他角色察觉
